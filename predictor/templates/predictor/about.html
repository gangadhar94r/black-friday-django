About This Project

Project Overview

This is a Black Friday Sales Prediction system that uses machine learning to forecast customer purchase amounts during the annual Black Friday sales event. Built as a Major Project, it implements both Random Forest and Linear Regression algorithms, allowing comparison between an ensemble method and a baseline approach.

Aim

To leverage data science and machine learning techniques to forecast sales trends for the Black Friday event, empowering retailers with actionable insights and accurate predictions that enable optimization of marketing strategies, inventory management, and resource allocation.

Dataset

The model is trained on the Black Friday Sales dataset containing 550,068 retail transactions. Each record includes customer demographics (age, gender, occupation, marital status), location (city category, years in city), product information (3 product categories), and the actual purchase amount.

Key Features
  1. Data Input Form — User-friendly form for customer and product details
  2. Real-time Prediction — Instant purchase amount prediction with spender tier
  3. Model Selection — Choose between Random Forest and Linear Regression
  4. Model Evaluation — View RMSE, MAE, and R² metrics for both models
  5. Visualization Tools — Interactive charts for model comparison and feature importance
  6. Responsive Design — Works on desktop, tablet, and mobile devices
Machine Learning Algorithms

Random Forest Regressor: An ensemble learning method that constructs multiple decision trees during training and outputs the mean prediction of individual trees. It handles non-linear relationships well and is robust against overfitting. Achieved R² = 0.65 on the test set.

Linear Regression: A baseline linear approach that models the relationship between input features and the target as a linear equation. While simpler, it provides interpretable coefficients and serves as a benchmark. Achieved R² = 0.15 on the test set.
